# 作曲マスター3Dバトル (仮称)

作曲先生・作曲ネコ・作曲フクロウ オト の3キャラが空中で戦うバトルゲーム。

## 構成

- **エンジン**: Vite + Three.js (ブラウザで動くプロトタイプ)
- **キャラ**: イラストから AI 3D生成 (Tripo3D / Meshy) → Blender微調整 → glTF/GLB
- **方向性**: スマブラ・あつ森系のスタイライズド3D
- **将来**: Unity移植を視野に入れる

## ディレクトリ構成

```
composer-battle/
├── assets/
│   └── characters/
│       ├── source/         # 元のイラスト画像
│       │   ├── cat_neko.png
│       │   ├── sensei.png
│       │   └── owl_oto.png
│       └── models/         # 生成・調整後の3Dモデル (.glb)
├── src/
│   ├── main.js             # Three.jsエントリポイント
│   ├── scene.js            # シーン構築
│   └── character.js        # キャラクター制御
├── index.html
├── package.json
└── README.md
```
